# Flamework Template (fork)

This is a fork of the official [flamework project template](https://github.com/rbxts-flamework/template). It includes fixes & additions that I personally find useful when starting a **roblox-ts + flamework** project.

## Usage

Run the following commands in a UNIX terminal:

```sh
# Change "myproject" to your project name
bunx degit xnlogical/flamework-template myproject
cd myproject
bun i
```
